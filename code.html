function reverseString(userString) {
let revString = [];

for (let index = userString.length - 1; index >= 0; index--) {
revString += userString[index];

}

return revString;
}

funciton displayString(revString){
document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;

document.getElementById("alert").classList.remove("invisible");

}

The Code

The getValue() function is a controller function that is first triggered by the click of the "Reverse Me" button. This function uses value property of the userString variable (the string the user inputs) to find the value of the string.

The .classList property is used to to access the "invisible" class assigned to the alert to element to prevent the alert from being triggered.

The reverseString() function is a logic function that takes an argument of userString. Because a string is an array of characters we tell JavaScript "This is an array; and because this is an array we have access to array functions" We can now add things to the array.

We reverse the string using a for loop. Every letter in a string has an index. If we write a for loop that starts at the end and loops back to the position (or index of 0) and record each of those characters, we would get a reversed String!

We make a decrementing for loop that grabs the id of the element we will be displaying our reversed string in. In this case, a paragraph element with an id of "msg"

displayString, is the third function, a display function that takes an argument of revString (our new reversed string). This function displays our reversed string through a temperate literal.

The second part of this function grabs the alert element by it's id alert and removes the inivsible class so that it will now display on the screen with the newly reversed string inside.
